# projeto\_final\_sprint\_5



Análise Exploratória de Dados - Veículos Usados
Este projeto é um aplicativo web interativo desenvolvido em Python utilizando Streamlit, projetado para realizar análises exploratórias em um conjunto de dados de anúncios de veículos usados.

Objetivo
O aplicativo permite visualizar e explorar dados de veículos à venda, ajudando usuários a identificar padrões e tendências relacionadas a preços, quilometragem e outros atributos dos veículos.

Funcionalidades
Histograma da Quilometragem: Gera um histograma interativo da coluna odometer (quilometragem), permitindo analisar a distribuição dos veículos com base na distância percorrida.

Gráfico de Dispersão Preço vs. Quilometragem: Cria um gráfico de dispersão relacionando o preço do veículo (price) com sua quilometragem (odometer), útil para visualizar a correlação entre essas duas variáveis.

Tecnologias Utilizadas
Python

Streamlit – para construção da interface web

Pandas – para manipulação dos dados

Plotly – para criação de gráficos interativos

Como Usar
Certifique-se de ter o arquivo vehicles.csv no diretório do projeto.

Execute o aplicativo com o comando:

streamlit run app.py
No navegador, clique nos botões para gerar os gráficos desejados.
